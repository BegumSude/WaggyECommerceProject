# Waggy E-Ticaret Sitesi 🐾

Bu proje, evcil hayvan ürünleri satışı yapan bir e-ticaret uygulamasıdır.

## 🚀 Özellikler
- Ürün listeleme
- Kullanıcı kaydı ve girişi
- Admin paneli
- Sipariş takibi

## 🛠️ Kullanılan Teknolojiler
- ASP.NET Core
- Entity Framework
- SQL Server
- HTML, CSS, JavaScript
